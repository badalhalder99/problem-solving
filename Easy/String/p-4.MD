### match() method in javaScript:

- match() হল JavaScript-এর একটি string method, যা একটি string-এর মধ্যে regex (regular expression) বা সাধারণ টেক্সট সার্চ করে এবং ম্যাচ করা ফলাফল return করে।

✅ Basic Syntax
string.match(regex);

### ✅ Case 1 — No RegEx (simple string):

const text = "Hello JavaScript";
console.log(text.match("Java"));

// ["Java"]
➡️ Returns the matched substring in an array

### ✅ Case 2 — Regex without g flag (returns detailed info)
const text = "hello world";
console.log(text.match(/o/));

// ["o", index: 4, input: "hello world", groups: undefined]
➡️ Returns first match + extra details like index

### ✅ Case 3 — Regex with g flag (returns all matches)
const text = "hello world";
console.log(text.match(/o/g));

// ["o", "o"]
➡️ Returns only matched values (no index)

### ✅ Case 4 — Case insensitive search (i flag)

const text = "JavaScript is Awesome";
console.log(text.match(/javascript/i));

// ["JavaScript"]

- i → ignore case
- g → global flag (match all)


### ✅ Case 5 — Count vowels in a string
const text = "Hello World";
const vowels = text.match(/[aeiou]/gi);

console.log(vowels);
// ["e", "o", "o"]

### ✅ When No Match Found?
console.log("abc".match(/z/));
// null


### ➡️ Always check before using length

const result = "abc".match(/z/);
console.log(result ? result.length : 0);
// 0

match() return	Array or null


✅ Summary

match() returns array of matches

Without g → first match + details
With g → all matches only

No match → null
Works best with regex
