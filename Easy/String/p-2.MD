- Again when you need to convert an array to string then you should use arr.join() method
- arr.join(separator)

- The join() method is an array method used to create and return a new string by concatenating all of the elements in an array.
- Combine array element/item into single string? This type of situation please use arr.join() method.

- separator (Optional): A string that separates each pair of adjacent elements of the array in the resulting string. If omitted, the array elements are separated by commas (,) by default. If an empty string ("") is provided, no separator is used.
Return Value:
- A new string containing all the array elements, joined by the specified separator. The original array remains unchanged.
Functionality:

- Concatenates elements: It takes each element of the array and combines them into a single string.
- Uses a separator: It inserts the specified separator string between each pair of adjacent elements.

- Default separator: If no separator is provided, it defaults to a comma (,).
- Returns a new string: The join() method does not modify the original array; instead, it returns a brand new string.

- Example is below:

const fruits = ["Banana", "Orange", "Apple", "Mango"];

// Using default separator (comma)
let text1 = fruits.join(); // "Banana,Orange,Apple,Mango"

// Using a space as a separator
let text2 = fruits.join(" "); // "Banana Orange Apple Mango"

// Using " and " as a separator
let text3 = fruits.join(" and "); // "Banana and Orange and Apple and Mango"

// Using an empty string as a separator
let text4 = fruits.join(""); // "BananaOrangeAppleMango"

const numbers = [1, 2, 3, 4];
let numString = numbers.join("-"); // "1-2-3-4"
